# Основы WEB-инжиниринга — подготовка к экзамену

Структура сделана под GitHub: каждый билет лежит в отдельном Markdown-файле. Внутри каждого файла один и тот же порядок: сам билет, пояснение по вопросам, готовая формулировка для ответа преподавателю и практика.

## Как учить

1. Открой билет и сначала прочитай блок **Пояснение** — он нужен, чтобы понять тему.
2. Потом выучи блок **Как лучше ответить преподавателю** — это короткая устная формулировка.
3. В конце обязательно прорешай **Практику**: код, SQL, алгоритм или схему.
